# Formulario Multi Paso - Casa Ronald McDonald Colombia

Este repositorio contiene el prototipo del formulario de donación multi-paso con dos variantes de diseño visual.

## Versiones Disponibles

### 1. Diseño Original (Clásico)
- **Archivo HTML**: `index.html`
- **Estilos**: `css/donar-online-multipasos.css`
- **Paleta**: Rojo sólido (#DA1A00), Azul (#4872AE) y Amarillo (#FFC829). Estilo corporativo limpio.

### 2. Diseño Rebranding (2026)
- **Archivo HTML**: `index-rebranding.html`
- **Estilos**: `css/donar-online-rebranding.css`
- **Paleta**: Crema (#FFFDE9), Negro, Lima (#9CD83E), Azul Royal (#1B43B2). Estilo moderno "Neo-Brutalism" / Pop.

## Estructura del Proyecto

```
/
├─ css/
│  ├─ donar-online-multipasos.css  # Estilos versión 1
│  └─ donar-online-rebranding.css  # Estilos versión 2
├─ index.html                      # Vista versión 1
├─ index-rebranding.html           # Vista versión 2
└─ README.md
```

## Cómo visualizar
Simplemente abre los archivos `.html` en tu navegador web para ver los diseños estáticos.
